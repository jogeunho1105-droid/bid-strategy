# 📊 투찰전략 분석 시스템 v2.10

낙찰이력을 분석해 업체 1·2·3의 추천 사정율과 산정 근거를 제공하는 버전입니다.
화면과 엑셀에는 최종 의사결정에 필요한 세 추천값만 표시합니다.

## 설치 및 실행

### 1단계 - Python 설치
https://python.org 에서 Python 3.11 다운로드 후 설치
(설치 시 "Add Python to PATH" 반드시 체크)

### 2단계 - 라이브러리 설치
터미널(명령 프롬프트)에서 아래 명령어 실행:
```
pip install streamlit pandas numpy openpyxl xlrd
```

### 3단계 - 앱 실행
```
streamlit run bid_app.py
```
브라우저에서 자동으로 열립니다 (http://localhost:8501)

## 외부 공유 (Streamlit Cloud 무료 배포)

### 필요한 것
- GitHub 계정 (무료)
- Streamlit 계정 (무료)

### 단계
1. GitHub에서 새 저장소 생성
2. bid_app.py / requirements.txt 업로드
3. share.streamlit.io 접속 → Deploy
4. 생성된 URL을 팀원에게 공유

### 비밀번호 설정 (배포자 페이지 보호)
Streamlit Cloud → Settings → Secrets 에서:
```
ADMIN_PWD = "원하는비밀번호"
```

## 사용 방법

### 배포자 (낙찰이력 관리)
1. 사이드바에서 "배포자 관리" 선택
2. 비밀번호 입력
3. 낙찰이력 xlsx 파일 업로드

### 일반 사용자 (전략 조회)
1. 사이드바에서 "투찰전략 분석" 선택
2. 입찰서류함 xls 파일 업로드
3. 자동 분석 결과 확인
4. 전략표 엑셀 다운로드

## 분석 방법론

| 구분 | 내용 |
|---|---|
| 업체 1 | 발주처 전체와 동일 분야의 양수·음수 전이패턴으로 다음 부호를 정하고, 직전 2건의 차이를 최신값에 더하거나 빼서 산정 |
| 업체 2 | 용역성격별 백테스트 최적모델로 산정한 중심 추천값 |
| 업체 3 | 동일 발주처·동일 분야를 우선한 관련 이력 최근 100건의 75% 분위수 |
| 산정 근거 | 사용한 이력 범위·표본 수·용역 분류·적용 모델 표시 |
| 사정율 흐름 차트 | 해당 발주처 전체 흐름과 해당분야 흐름 2개만 단순 비교 |

## 사용 절차

1. 배포자 관리에서 낙찰이력 파일을 업로드합니다.
2. 투찰전략 분석 화면에서 나라장터 입찰서류함 파일을 업로드합니다.
3. 요약표의 `업체1추천`, `업체2추천`, `업체3추천`을 확인합니다. 각 근거는 `비고`에서 업체별 줄바꿈으로 표시됩니다.
4. 건별 상세의 사정율 흐름 차트를 확인한 뒤 최종 투찰값을 결정합니다.
