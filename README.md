# 📊 투찰전략 분석 시스템 v2.7

기존 `bid-strategy` 앱에 백테스트 기반 개선모델을 추가한 버전입니다.
기존 ①패턴, ②유사표본, ③트렌드, 3포인트 전략은 유지하고, 낙찰이력
31,586건 기준으로 용역성격별 가장 성과가 좋았던 모델의 추천값을 함께 제공합니다.

## 설치 및 실행

### 1단계 - Python 설치
https://python.org 에서 Python 3.11 다운로드 후 설치
(설치 시 "Add Python to PATH" 반드시 체크)

### 2단계 - 라이브러리 설치
터미널(명령 프롬프트)에서 아래 명령어 실행:
```
pip install streamlit pandas numpy openpyxl xlrd
```

### 3단계 - 앱 실행
```
streamlit run bid_app.py
```
브라우저에서 자동으로 열립니다 (http://localhost:8501)

## 사용 방법

### 배포자 (낙찰이력 관리)
1. 사이드바에서 "배포자 관리" 선택
2. 비밀번호 입력
3. 낙찰이력 xlsx 파일 업로드

### 일반 사용자 (전략 조회)
1. 사이드바에서 "투찰전략 분석" 선택
2. 입찰서류함 xls 파일 업로드
3. 자동 분석 결과 확인
4. 전략표 엑셀 다운로드

## 분석 방법론

| 구분 | 내용 |
|---|---|
| 개선추천 | 용역성격별 백테스트 최적모델로 추천 중심값 산정 |
| 추천 사정율 하한/상한 | 모델별 평균오차와 목표 적중률 60/70/80% 설정에 따라 추천구간 산정 |
| ①패턴 | 발주처 전체 이력 — 트렌드·자기상관·가중치 최적화 |
| ②유사표본 | 유사 용역명·기초금액 ±50% 범위 낙찰이력 |
| ③트렌드 | 발주처 최근 흐름 — 최신 건에 높은 가중치 |
| 기존 권장구간 | 3가지 값의 평균 ± 표준편차×0.5 |

## 개선모델 적용 기준

| 용역성격 | 기본 적용모델 |
|---|---|
| PD | 용역성격+금액구간 최근20건 |
| VLF | 가중혼합 |
| 콘크리트 | 용역성격 유사패턴 최근10건 |
| 건설사업관리 | 발주처+용역성격 최근20건 |
| 설계 | 가중혼합 |
| 진단기타 | 발주처+용역성격 최근20건 |
| 광학 | 평균혼합 |
| 기타 | 발주처+용역성격 유사패턴 최근5건 |
| 감리 | 발주처+용역성격 최근20건 |
| 초음파 | 발주처 최근20건 |

## 사용 절차

1. 배포자 관리에서 낙찰이력 파일을 업로드합니다.
2. 사이드바에서 개선 추천구간 목표 적중률을 60%, 70%, 80% 중 선택합니다.
3. 투찰전략 분석 화면에서 나라장터 입찰서류함 파일을 업로드합니다.
4. 요약표의 `개선추천`, `추천 사정율 하한`, `추천 사정율 상한`, `적용모델`을 우선 확인합니다.
5. 건별 상세에서 기존 ①②③ 분석값과 개선추천의 차이를 검토한 뒤 최종 투찰값을 결정합니다.
